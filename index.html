安装说明

macOS 安装注意事项

macOS 安装可能会出现报错。如果报此问题，进入到安装文件所在目录，执行以下命令后再进行安装：

# 1. 赋予核心二进制文件可执行权限
chmod +x DevTools.app/Contents/MacOS/app

# 2. 彻底移除 macOS 的隔离属性
xattr -cr DevTools.app

Windows 和 Linux 用户：正常安装即可
